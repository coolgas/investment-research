BLOCKCHAIN ACADEMIC PAPER VERIFICATION AND FRAUD DETECTION - RESEARCH NOTES
=========================================================================
Date: 2026-01-05
Researcher context: TY, physics PhD student. Core interest: decentralized trust/verification
mechanisms for academic papers using crypto rewards and FULL anonymity for verifiers.

=========================================================================
ACTIVE PROJECTS (LIVE / OPERATIONAL)
=========================================================================

1. ResearchHub / ResearchCoin (RSC)
URL: https://www.researchhub.com / https://www.researchhub.foundation/research-coin
Blockchain: Ethereum (ERC-20) + Base L2
Token: ResearchCoin (RSC)
Anonymity: Partial. Reviews on ResearchHub are typically public with real names/
ORCID linked. Some features may allow pseudonymous interaction but full
anonymity is NOT a core design goal.
Status: ACTIVE. Founded 2020, backed by Coinbase CEO Brian Armstrong.
How it works: Decentralized platform where scientists earn RSC for publishing
research, doing peer reviews, and other contributions. Users can post papers,
comment, upvote, and peer review. Rewards distributed algorithmically based on
impact. The ResearchHub Journal (launched 2024) pays peer reviewers ~$150/RSC
equivalent per review - reported by Nature Dec 2024 (doi:10.1038/d41586-024-04027-4).
RSC has real market value traded on exchanges. NOT anonymous - it's a public
reputation system. Source: Nature News, DAOTimes analysis, Dapp.Expert.
Key limitation for TY: No anonymity for reviewers. Your identity/reputation is
the point.

2. Bloxberg
URL: https://bloxberg.org/
Blockchain: Custom permissioned Layer 1 (since 2019), evolving to Layer 2 on Ethereum
Token: None native (uses reputation, no cryptocurrency rewards mentioned)
Anonymity: No. Consortium-governed, institutions are known.
Status: ACTIVE. Consortium of 57 research institutes across 5 continents,
representing 63,000+ scientists. 83,000+ wallets, 25M+ transactions.
How it works: "Blockchain for science" - public-permissioned L1 built by and for
science. Provides infrastructure for scientific data verification, certification,
and decentralized identity. Issues verifiable credentials. Does NOT have a token
reward mechanism for peer review. Focused on data integrity and certification
(e.g., NFT-based research objects called ROCs). Governed by bloxberg Council.
Source: bloxberg.org.

3. Blockcerts
URL: https://www.blockcerts.org/
Blockchain: Bitcoin (originally), now multiple chains
Token: None
Anonymity: N/A (credential verification system)
Status: ACTIVE. Open standard developed by MIT Media Lab and Learning Machine.
How it works: Open standard for issuing and verifying blockchain-based academic
credentials. Not a peer review or fraud detection system. Focused on diploma/
certificate verification. Uses blockchain to make credentials tamper-proof and
self-sovereign. No token rewards. Source: blockcerts.org.

4. Octopus
URL: https://int.octopus.ac/
Blockchain: None (not blockchain-based)
Token: None
Anonymity: Partial (ORCID-based)
Status: ACTIVE. Funded by UKRI (UK government research funder).
How it works: A new publishing platform for scholarly research. NOT blockchain.
Focuses on publishing research in full detail, enabling peer review and quality
assessment. No crypto rewards. Included for completeness - it's a reform effort
but not relevant to blockchain verification. Source: octopus.ac.

PROJECTS WITH ANONYMITY FEATURES (MOST RELEVANT TO TY)

5. Ants-Review
URL: https://arxiv.org/abs/2101.09378 / https://github.com/naszam/ants-review
Blockchain: Ethereum (proposed)
Token: Proposed incentive token (not specified as launched)
Anonymity: YES - designed for anonymous open peer review. Uses privacy
mechanisms for reviewer anonymity.
Status: RESEARCH PROPOSAL / DEFUNCT? Academic paper (Jan 2021), GitHub repo exists
but last activity likely 2021. Never launched as live platform.
How it works: Protocol for incentivized open peer reviews on Ethereum. Rewards
scientists with tokens for peer review. Designed to support anonymity for
reviewers while providing cryptographic guarantees of review quality. Described
as a "privacy-oriented protocol." Smart contract-based bounty system where
reviewers submit reviews pseudonymously and earn tokens. Key paper by Bianca
Trovo and Nazzareno Massari, also published as Springer chapter (doi:10.1007/
978-3-030-71593-9_2). Source: arXiv, Springer, GitHub.
Key relevance for TY: This is the closest match to what TY is asking about -
anonymous peer review with crypto rewards. But it's a research proposal, not a
live system.

6. BARIT (Blockchain-based Anonymous Reviewer Incentive Token)
URL: https://ieeexplore.ieee.org/abstract/document/10831118
Blockchain: Proposed (not specified which)
Token: BARIT token (proposed)
Anonymity: YES - anonymous reviewer incentives are the core design goal
Status: RESEARCH PAPER (published 2024/2025). Master's thesis by Pratiksha at
University of Miami/WSL.
How it works: A case study on a blockchain-based anonymous reviewer incentive
token. Designed specifically to allow reviewers to remain anonymous while
earning token rewards for quality peer reviews. Uses blockchain to create
anonymous review tokens. Source: IEEE SMC 2024, thesis PDF.
Key relevance: Another proposal, not live. Directly addresses TY's core question
about anonymous rewarded review.

7. BeerReview
URL: https://arxiv.org/abs/2405.20220
Blockchain: Proposed (blockchain-enabled, not specified which)
Token: No token mentioned
Anonymity: Implicit (addresses plagiarism concerns, enables secure participation)
Status: RESEARCH PAPER + ALPHA TESTING. Published May 2024. Authors claim alpha
testing completed.
How it works: Blockchain-enabled peer review platform addressing plagiarism,
malicious attacks, and unauthorized data access. Enables experts to participate
without concerns about IP theft. Smart contracts manage the review workflow.
Source: arXiv:2405.20220 by Jin, Zhou, Tang, Yu, Xu, Liu.

TOKEN-CURATED REGISTRIES (TCRs) FOR ACADEMIC PAPERS
=========================================================================

14. Token-Curated Registry in a Scholarly Journal (TCR concept applied)
URL: https://arxiv.org/abs/2005.11534 / https://onlinelibrary.wiley.com/doi/full/
10.1002/leap.1302
Status: ACADEMIC PAPER (published in Learned Publishing, 2020)
Summary: Proposes applying TCR mechanism to scholarly journals. A community
curates a list of high-quality papers by staking tokens. Reviewers who correctly
identify quality papers are rewarded; those who stake on low-quality papers lose
their stake. This creates a decentralized quality control mechanism. Source:
Jan Erik Frantsvag, "Token-curated registry in a scholarly journal: Can
blockchain support journal communities?" Learned Publishing 2020.

15. Token-Curated Registry with Citation Graph (CitedTCR)
URL: https://arxiv.org/abs/1906.03300
Status: ACADEMIC PAPER (2019)
Summary: Kensuke Ito and Hideyuki Tanaka propose incorporating expertise into
TCRs using citation graphs. Uses Personalized PageRank (PPR) to assign curators
based on their citation network position. Multi-task peer-prediction mechanism
for reward computation. Specifically designed for technical content like
academic papers and patents. Source: arXiv:1906.03300.

16. ChainScore Labs - TCR Guides for Scientific Data
URL: https://chainscorelabs.com/guides/decentralized-science-desci/incentive-
system-design/launching-a-token-curated-registry-for-scientific-datasets
Status: CONSULTING / EDUCATIONAL CONTENT (2025-2026 era)
Summary: ChainScore Labs provides guides on launching TCRs for scientific
datasets and research reputation. Not an implementation itself, but practical
guidance for DeSci projects wanting to implement TCRs. Source: chainscorelabs.com.

KEY TAKEAWAYS FOR TY
=========================================================================

1. NO live project fully matches TY's requirements (anonymity + crypto rewards
   + fraud detection). This is a gap in the market.

2. Closest approximations:
   - ResearchHub/RSC: Live, has token rewards, but NOT anonymous. Reputation-
     based system where identity is public.
   - Ants-Review: Proposed protocol that matches all criteria (anonymity +
     rewards) but was never implemented as a live platform.
   - BARIT: Academic proposal for anonymous reviewer tokens. Not live.
   - PubPeer: Anonymous fraud detection platform but NO crypto rewards and
     NOT on blockchain.

3. Anonymity mechanisms available (but not deployed in this context):
   - zk-SNARKs/STARKs: Can prove expertise without revealing identity
   - Ring signatures: Can make reviewer identity indistinguishable within a set
   - Anonymous credentials: Can prove qualifications anonymously
   These technologies exist but have NOT been packaged into a working academic
   verification platform.

4. TCR concept for academic papers exists in theory (Frantsvag 2020, Ito &
   Tanaka 2019) but NO live implementation has been identified.

5. DeSci movement is growing (Bloxberg, ResearchHub, Ethereum Foundation
   support) but anonymous fraud detection specifically is not addressed by
   major players.

6. Gap analysis: A platform combining (a) anonymous verification using ZK
   proofs, (b) token rewards for catching fraud, and (c) on-chain
   reproducibility checking would be novel. No one has built this.
